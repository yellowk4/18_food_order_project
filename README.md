### Installation

다음 단계를 따라 프로젝트를 설치하십시오:

1. 저장소를 클론합니다: `git clone https://github.com/yellowk4/18_food_order_project.git`
2. 디렉토리로 이동합니다: `cd 18_food_order_project`
3. 의존성을 설치합니다: `npm install`
4. 프로젝트를 실행합니다: `npm start or npm run dev`
